# 관리자 상세 운영 매뉴얼

## 1. 빠른 시작
1. 관리자 로그인 후 대회를 선택합니다.
2. 종별을 확인하고 필요한 세부종목을 등록합니다.
3. 선수 등록 또는 일괄 업로드로 명단을 준비합니다.
4. 세부종목 운영 화면에서 출전선수등록과 레인 배정을 마칩니다.
5. 점수 입력 후 한 게임이 끝나면 순위 반영을 눌러 최신 순위를 계산합니다.

## 2. 대회 준비
- 종별은 실제 운영 단위와 같게 맞춥니다.
- 세부종목 생성 시 경기일, 게임 수, 시작/끝 레인, table 이동값을 반드시 확인합니다.
- 선수 등록 후 번호, 이름, 시군, 소속, 손, 팀조가 맞는지 점검합니다.

## 3. 레인 운영
- 출전선수등록 탭에서 실제 출전 선수만 선택합니다.
- 레인 배정 탭에서 드래그앤드롭으로 배정합니다.
- 같은 레인 안에서도 순서를 바꿀 수 있으며, 변경 후 저장해야 유지됩니다.
- 랜덤 배정 후에도 필요하면 수동으로 조정한 뒤 다시 저장합니다.

## 4. 점수 입력과 순위 반영
- 현재 게임과 레인을 확인한 뒤 점수를 입력합니다.
- 선수별 저장 또는 레인 전체 저장을 사용합니다.
- 점수 저장만으로는 순위가 갱신되지 않습니다.
- 한 게임 입력이 끝난 뒤 순위 반영 버튼을 눌러 세부순위와 종합순위를 갱신합니다.

## 5. 자주 발생하는 상황
- 순위가 예전 값으로 보이면 순위 반영을 다시 실행합니다.
- 같은 레인 순서가 저장되지 않으면 레인 배정 후 저장을 다시 확인합니다.
- 잘못 입력한 점수는 같은 선수 점수를 다시 저장하면 최신 값으로 덮어씁니다.
