# PRD: 경기도체육회 볼링대회 성적 관리 SaaS

작성일: 2026-03-04
버전: 0.1 (Firebase 기반 설계 반영)

## 1. 목표
대회별로 선수등록부터 점수 입력, 레인 자동/수동 배정, 실시간 순위 집계까지 한 번에 처리하는 웹 SaaS를 만든다.

타깃:
- 경기도체육회 볼링협회
- 대회 운영 스태프(관리자)
- 대회 관람자/선수(공개 조회)

## 2. 핵심 사용자
- 일반 사용자(로그인 없음)
  - 대회 목록을 조회하고 카드형으로 클릭해 상세 성적 확인
  - 종별/세부종목별 순위 및 실적 조회
  - 전체 종합 성적 조회
  - 키워드 검색 및 필터
- 관리자
  - 대회 관리(CRUD)
  - 종별/세부종목 설정
  - 선수 등록, 선발성(시도/소속/번호/성명/손)을 포함한 명단 관리
  - 레인 배정(랜덤/드래그드롭 수동 조정)
  - 게임 점수 입력(게임 총점)
  - 실시간 순위 반영

## 3. 핵심 기능
### 일반 사용자 화면
1. 대시보드
   - 대회 카드 목록(카드 형)
   - 키워드 검색
   - 연도/지역/상태 필터
2. 대회 상세
   - 종별(예: 초등부 U-10 남자/여자, 중등부 남자/여자, ...)
   - 세부종목(개인전, 2인조, 3인조, 4인조, 5인조, 개인종합)
   - 세부종목 선택 시 해당 성적표 표시
3. 성적표
   - 컬럼: 순위, 시도(게임 참여 횟수), 소속, 번호, 성명, 1G~6G, 합계, 평균, 핀차이
4. 전체 종합 성적
   - 해당 대회 모든 세부종목의 합산 점수
   - 컬럼: 순위, 시도, 소속, 번호, 성명, 1G~6G, 합계, 평균, 핀차이, 게임수

### 관리자 화면
1. 대회 관리(조회/생성/수정/삭제)
2. 대회 환경설정
   - 대회명
   - 대회기간
   - 세부종별 설정(추가/수정/삭제)
   - 세부종목 등록/삭제(개인전, 2인~5인, 개인종합)
   - 세부종목별 경기 날짜
   - 사용 레인(시작~끝)
3. 세부종목 설정
   - 사용 레인(시작~끝)
   - Table 이동수(드롭다운 1~20, 음수 허용)
4. 선수 관리
   - 등록 항목: 팀조(A/B), 시군, 소속(학교), 선수번호(자동), 성명, 손(왼손/오른손)
   - 선수는 대회 단위로만 유효
5. 레인 배정
   - 랜덤 배정(대회 등록 선수 수 기준 2~4명/레인 규모로 자동)
   - 수동 배정(드래그&드롭으로 선수 스왑)
6. 경기 진행 및 점수 입력
   - 게임 종료 후 다음 게임 자동 레인 이동표 생성
   - 예: Table 이동수 +2 → 게임1 레인 1, 게임2 레인 3, 게임3 레인 5
   - 게임 당 레인별 선수명단(소속, 성명, 선수번호) 표시
   - 각 게임 점수 입력 시 실시간 순위 반영

## 4. 비즈니스 제약
- 매출/구독 모델은 현재 제외(요금제 미정)
- 첫 단계는 기능적 완성(대회 운영 성능, 안정성 우선)

## 5. 요구사항 우선순위
### P0 (MVP)
- 대회 생성/조회
- 종별/세부종목 CRUD
- 선수 등록 및 대회별 데이터 분리
- 랜덤 레인 배정
- 게임 점수 입력(총점)
- 실시간 점수 반영 + 순위 계산(세부종목/종합)
- 레인 이동 및 다음 게임명단 표시

### P1
- 관리자 인증 및 권한
- 수동 드래그앤드롭 배정
- Firebase Rules 기반 보안 강화
- 필터 성능 개선/캐싱/대회 검색 UX 정교화

### P2
- 알림
- 경기장 레이아웃별 뷰 모드
- CSV/엑셀 내보내기

## 6. 데이터 표기 규칙
- 합계: 입력된 게임 점수 합계(총합)
- 평균: 합계 / 시도(게임 참여 횟수)
- 핀차이: 1위 합계 - 해당 선수 합계

## 7. 수용 기준
- 같은 대회 내에서만 선수·점수·배정이 공유되고 다른 대회와 격리됨
- 세부종목 점수 입력이 변경되면 세부종목 성적표와 전체 종합 성적이 즉시 업데이트됨
- Table 이동수 음수/양수 모두 동작
 - 사용 레인 범위 밖 선수 배정/점수 입력이 불가하도록 제어

## 운영 보안
- 관리자 로그인 필수: `/admin` 진입은 Firebase 인증 세션으로 제한
- 관리자 권한: 환경변수 `FIREBASE_ADMIN_EMAILS` 또는 Firebase 커스텀클레임 `admin=true` 허용
